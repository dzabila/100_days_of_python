­ЪД« Python Рђћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў
1. Primitive Data Types (рЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў)

Python-рЃерЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў (primitive) рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃўрЃљ:

рЃбрЃўрЃърЃў	рЃљрЃдрЃгрЃћрЃарЃљ	рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў
int	рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў	5, -10, 0
float	рЃљрЃЌрЃгрЃўрЃџрЃљрЃЊрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў	3.14, -2.5
bool	рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў	True, False
str	рЃбрЃћрЃЦрЃАрЃбрЃў (рЃАрЃбрЃарЃўрЃюрЃњрЃў)	"Hello", 'Python'

­ЪЊў рЃћрЃА рЃбрЃўрЃърЃћрЃЉрЃў рЃЦрЃЏрЃюрЃўрЃљрЃю рЃДрЃЋрЃћрЃџрЃљ рЃАрЃ«рЃЋрЃљ рЃарЃЌрЃБрЃџрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА (рЃЏрЃљрЃњ. list, dict) рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃА.

2. TypeError

TypeError рЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃЏрЃљрЃерЃўрЃю, рЃарЃЮрЃфрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ рЃЋрЃфрЃЊрЃўрЃџрЃЮрЃЉрЃЌ рЃљрЃарЃљрЃАрЃгрЃЮрЃа рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљрЃА.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:

print("Age: " + 20)      # РЮї TypeError (string + int)
print("Age: " + str(20)) # РюЁ рЃАрЃгрЃЮрЃарЃў рЃњрЃќрЃљ


рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў: рЃбрЃўрЃърЃўрЃА рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ (int(), str(), float() рЃЊрЃљ рЃљ.рЃе.)

3. Type Checking (рЃбрЃўрЃърЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ)

рЃбрЃўрЃърЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ type() рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ:

x = 10
print(type(x))  # <class 'int'>

y = "Hello"
print(type(y))  # <class 'str'>


­ЪЉЅ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃўрЃА рЃњрЃљрЃАрЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃџрЃљрЃЊ, рЃарЃЮрЃфрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃЋрЃўрЃфрЃЮрЃЊрЃћрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃбрЃўрЃърЃў.

4. Type Conversion (рЃбрЃўрЃърЃўрЃА рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ)

рЃбрЃўрЃърЃўрЃА рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃљрЃЊ рЃњрЃљрЃЊрЃљрЃДрЃЋрЃљрЃюрЃљрЃА.

рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ	рЃљрЃдрЃгрЃћрЃарЃљ	рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў
int(x)	рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃЏрЃЌрЃћрЃџ рЃарЃўрЃфрЃ«рЃЋрЃљрЃЊ	int("5") Рєњ 5
float(x)	рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃљрЃЌрЃгрЃўрЃџрЃљрЃЊрЃљрЃЊ	float("3.14") Рєњ 3.14
str(x)	рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃАрЃбрЃарЃўрЃюрЃњрЃљрЃЊ	str(10) Рєњ "10"
bool(x)	рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА True/False рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃЊ	bool(0) Рєњ False, bool(1) Рєњ True
5. Mathematical Operators (рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў Python-рЃерЃў)

Python-рЃерЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃќрЃћ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў:

рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў	рЃљрЃдрЃгрЃћрЃарЃљ	рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў	рЃерЃћрЃЊрЃћрЃњрЃў
+	рЃЏрЃўрЃЏрЃљрЃбрЃћрЃЉрЃљ	5 + 2	7
-	рЃњрЃљрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃљ	5 - 2	3
*	рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ	5 * 2	10
/	рЃњрЃљрЃДрЃЮрЃцрЃљ (float)	5 / 2	2.5
//	рЃЏрЃЌрЃћрЃџрЃќрЃћ рЃњрЃљрЃДрЃЮрЃцрЃљ	5 // 2	2
%	рЃюрЃљрЃерЃЌрЃў	5 % 2	1
**	рЃ«рЃљрЃарЃўрЃАрЃ«рЃерЃў рЃљрЃДрЃЋрЃљрЃюрЃљ	2 ** 3	8
6. Number Manipulation (рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃЌрЃљрЃю рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў)

Python рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃќрЃћ рЃњрЃљрЃюрЃЋрЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃЮрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў:

x = 10
x += 5   # рЃўрЃњрЃўрЃЋрЃћрЃљ рЃарЃљрЃф x = x + 5
x -= 2   # рЃўрЃњрЃўрЃЋрЃћрЃљ рЃарЃљрЃф x = x - 2


рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:

round(3.14159, 2)   # 3.14 Рђћ рЃљрЃЏрЃарЃњрЃЋрЃљрЃџрЃћрЃЉрЃА 2 рЃфрЃўрЃцрЃарЃљрЃЏрЃЊрЃћ
abs(-7)             # рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА 7 Рђћ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ
pow(2, 3)           # 8 Рђћ рЃўрЃњрЃўрЃЋрЃћрЃљ рЃарЃљрЃф 2 ** 3

7. f-Strings (рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃбрЃарЃўрЃюрЃњрЃћрЃЉрЃў)

f-string рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃбрЃарЃўрЃюрЃњрЃерЃў рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃЕрЃљрЃЋрЃгрЃћрЃарЃЮрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

name = "Nika"
age = 20
print(f"My name is {name} and I am {age} years old.")
# рЃерЃћрЃЊрЃћрЃњрЃў: My name is Nika and I am 20 years old.


рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃў:

print(f"2 + 3 = {2 + 3}")

­ЪДа рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ

Python-рЃерЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃбрЃўрЃърЃћрЃЉрЃўрЃљ: int, float, bool, str

TypeError рЃЕрЃюрЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃфрЃљ рЃЋрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃЌ рЃерЃћрЃБрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃћрЃџ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃћрЃЉрЃќрЃћ

type() рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃбрЃўрЃърЃА

int(), float(), str() рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃљрЃю рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃбрЃўрЃърЃА

рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃћрЃю рЃарЃЮрЃњрЃЮрЃарЃф рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃљрЃерЃў

f-string рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃњрЃќрЃљ рЃбрЃћрЃЦрЃАрЃбрЃерЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ
