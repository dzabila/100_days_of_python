­ЪДЕ 1. Module (рЃЏрЃЮрЃЊрЃБрЃџрЃў)

рЃЏрЃЮрЃЊрЃБрЃџрЃў Рђћ рЃћрЃА рЃљрЃарЃўрЃА Python рЃцрЃљрЃўрЃџрЃў (.py), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЎрЃЮрЃЊрЃА: рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃА, рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃА рЃљрЃю рЃЎрЃџрЃљрЃАрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃАрЃ«рЃЋрЃљрЃњрЃљрЃюрЃљрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ.
рЃўрЃњрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃЎрЃЮрЃЊрЃў рЃЊрЃљрЃДрЃЮ рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљрЃЊ рЃЊрЃљ рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃљрЃа рЃЊрЃљрЃгрЃћрЃарЃЮ рЃћрЃарЃЌрЃўрЃЊрЃљрЃўрЃњрЃўрЃЋрЃћ рЃџрЃЮрЃњрЃўрЃЎрЃљ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

# greetings.py
def say_hello(name):
    print(f"Hello, {name}!")


рЃАрЃ«рЃЋрЃљ рЃцрЃљрЃўрЃџрЃерЃў:

import greetings
greetings.say_hello("Nika")


­ЪЉЅ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃўрЃЊрЃћрЃљрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃљ рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃљрЃџрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.

­Ъј▓ 2. Random Module

random рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃљрЃарЃЕрЃћрЃЋрЃљрЃюрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљрЃерЃў.

рЃўрЃЏрЃърЃЮрЃарЃбрЃў:

import random


рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:

random.random()       # рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float-рЃА 0.0 рЃЊрЃљ 1.0 рЃерЃЮрЃарЃўрЃА
random.randint(1, 10) # рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃЌрЃћрЃџ рЃарЃўрЃфрЃ«рЃЋрЃА 1-рЃЊрЃљрЃю 10-рЃЏрЃЊрЃћ
random.choice(list)   # рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃАрЃўрЃўрЃЊрЃљрЃю
random.shuffle(list)  # рЃљрЃарЃћрЃЋрЃА рЃАрЃўрЃљрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃљрЃЊ
random.uniform(1, 5)  # рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float-рЃА 1 рЃЊрЃљ 5 рЃерЃЮрЃарЃўрЃА

­ЪЊд 3. Import Module in File

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃњрЃќрЃўрЃЌ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ:

рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў	рЃљрЃдрЃгрЃћрЃарЃљ	рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў
import module	рЃўрЃЏрЃърЃЮрЃарЃбрЃљрЃЋрЃА рЃЏрЃЌрЃћрЃџ рЃЏрЃЮрЃЊрЃБрЃџрЃА	import math
import module as m	рЃљрЃФрЃџрЃћрЃЋрЃА рЃЏрЃћрЃбрЃАрЃљрЃ«рЃћрЃџрЃА	import random as r
from module import func	рЃўрЃЏрЃърЃЮрЃарЃбрЃљрЃЋрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА	from math import sqrt
from module import *	рЃўрЃЏрЃърЃЮрЃарЃбрЃљрЃЋрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА	from math import * (рЃљрЃа рЃљрЃарЃўрЃА рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃў)
­Ъј» 4. Some Random Functions (random module-рЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃќрЃћ)
рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ	рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ	рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў	рЃерЃћрЃЊрЃћрЃњрЃў
random.random()	рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў float 0РђЊ1 рЃерЃЮрЃарЃўрЃА	random.random()	0.478
random.randint(a,b)	рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃў	random.randint(1,6)	3
random.uniform(a,b)	float	random.uniform(1,5)	3.25
random.choice(seq)	рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА	random.choice(['a','b','c'])	'b'
random.shuffle(list)	рЃљрЃарЃћрЃЋрЃА рЃАрЃўрЃљрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃљрЃЊ	РђЊ	[3,1,2]
random.sample(population, k)	рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ k рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА	random.sample(range(10), 3)	[1, 7, 4]
­ЪДЙ 5. List (рЃАрЃўрЃљ)

List Рђћ рЃљрЃарЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљрЃА.
рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃА.

рЃерЃћрЃЦрЃЏрЃюрЃљ:

numbers = [1, 2, 3, 4]
mixed = [1, "Hello", True, 3.14]


рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў:

рЃфрЃЋрЃљрЃџрЃћрЃЉрЃљрЃЊрЃўрЃљ (mutable)

рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃћрЃЉрЃўрЃЌ

рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃџрЃБрЃърЃўрЃЌ рЃњрЃљрЃЊрЃљрЃЋрЃџрЃљ

­ЪћД 6. Popular List Methods
